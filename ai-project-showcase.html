The Objective & Baseline

The primary goal of this project was to analyze historical data of 100,000 engineering students to uncover the hidden patterns driving successful campus placements.

By applying data science techniques, we aimed to transition from anecdotal assumptions to data-driven truths regarding how internships, backlogs, college tiers, and specific skills affect employability.

Branch Performance Baseline

We first established the baseline placement rates across different engineering branches. Computer Science Engineering (CSE) led the placements with a 71.3% success rate, indicating a higher industry demand for software roles compared to core branches like Mechanical or Civil.

Key Insights & Variables

  • The Internship Effect: Internships proved to be massive accelerators. Each internship added approximately ₹1.15 LPA to the average salary and boosted placement probability by 4–5%.
  • The Backlog Penalty: Academic consistency matters. Every additional backlog reduced placement chances by a stark 3%.
  • College Tier Impact: We found a massive 23% gap in placement rates between Tier-1 (83.1%) and Tier-3 (60.1%) colleges.
  • The Skill Myth: Interestingly, core technical skills (like Data Structures & Algorithms) were statistically similar across all tiers. This proved that the Tier-1 advantage is heavily influenced by institutional networking and brand value rather than pure technical superiority.

Predictive Machine Learning Models

Beyond retrospective analysis, we built predictive machine learning models to provide actionable foresight for current students.

  • Placement Classification: A Logistic Regression model was trained to predict the binary outcome (Placed/Not Placed). It achieved an accuracy of 69.8%, proving reliable for baseline estimations.
  • Salary Estimation: A Random Forest Regressor was developed to estimate expected CTC based on a student's numerical profile, yielding a robust R² score of 0.759.
  • Strongest Predictors: Feature importance extraction confirmed that CGPA is the absolute highest importance feature, closely followed by General Aptitude Scores and Machine Learning knowledge.

Practical Applications

The models and findings from this analysis pave the way for powerful, student-centric applications designed to maximize career success:
